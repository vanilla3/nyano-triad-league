# NYTL Battle Board 見た目ギャップ整理 v1（参照画像 → 実装タスク）

対象: Mint UI の Match（バトル盤面 + 周辺 HUD）

このドキュメントは、参照画像の雰囲気（“かわいらしいモバイルゲーム UI”）に近づけるために、
**「どこが足りないか」→「何を作れば埋まるか」** をタスクに落とすためのメモです。

---

## 1) 参照画像の要素（言語化）

### 1-1. 形状（シルエット）

- パネル/ボタン/セルが **“厚みのあるガラス”** として統一されている
- 外側に太めの縁（リム）、内側に細い縁（インナーリム）
- 角はやや大きめの丸み（スマホ想定で“安心”する曲率）

### 1-2. 質感（マテリアル）

- 透明感（backdrop-filter 相当）
- 斜めのシーン（specular highlight）が必ずある
- 極薄のノイズ（均一なベタ塗りを避ける）
- シャドウが柔らかく、色が薄い（黒ではなく、環境色を混ぜる）

### 1-3. 画面の“密度”

- 背景に雲/きらめき/肉球などが薄く入り、画面が寂しくならない
- ただし主役（盤面/カード）より目立たない

---

## 2) 現状の Mint UI で起きがちな不足

※以下は「よくある不足」なので、実装の現状に合わせて調整してください。

### 2-1. 押せる/押せないが近い

- 影/縁/ハイライトが薄いと、ボタンと飾りパネルの差が出にくい
- 押下時の“沈み込み”が弱いと、触り心地が平坦になる

### 2-2. 盤面セルが“トレイ”に見えない

- 空セルが平面に見え、カードが「置ける場所」だと直感しにくい
- セルごとに“凹み”と“縁”が弱い

### 2-3. 光の方向が統一されていない

- 各 UI のハイライト方向がバラつくと、手作り感が出る

---

## 3) 改善の設計方針（最短で効く順）

### P0: 盤面セルのマテリアルを統一して「置き台感」を出す

- セルを 3 層で描く
  1. ベース（薄いグラデ）
  2. 凹み（inset shadow + 底の暗さ）
  3. 斜めシーン（光の筋）

### P0: ボードトレイを “一段上の板” として見せる

- 盤面全体の外枠に厚みを持たせ、内側にさらに凹みを作る
- 端のハイライト（上/左）と影（下/右）を固定

### P1: 押下/選択の共通言語を作る

- ボタン、セル、カードで同じ「押した感」
  - 通常: 影あり
  - hover: 少し浮く
  - press: 少し沈む
  - selected: 縁が太く/明るく

### P2: “密度” は背景レイヤーで足す

- きらめき/肉球/雲は 3〜8% の薄レイヤーで足す
- 粒子は `data-vfx` で段階制御
