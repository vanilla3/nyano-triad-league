# NYTL Battle Board UI 品質ガイド（桜井目線） v1

対象: `Match`（Mint UI）における「バトル盤面」まわり

目的: 参照画像の“かわいさ/ゲーム感”を目指しつつ、
**「迷わない」「気持ちいい」「読みやすい」** を絶対条件にして、盤面の体験品質を底上げする。

---

## 0) 前提：UI は“飾り”ではなく、遊びのストレスを減らす装置

- 盤面 UI は「情報の提示」と「操作の導線」を担う。
- かわいさ/質感は重要だが、**読み取り速度** と **誤操作率** を落とすなら価値がない。

このガイドは「見た目の豪華さ」よりも、
**“触った瞬間にゲームが返事をする”** という体験を最優先に設計する。

---

## 1) 桜井目線のコア（盤面 UI に効く 7 つの原則）

### 原則 1: 次の一手が 0.5 秒で分かる

- 何をすべきか（カードを選ぶ / 置き場所を選ぶ）が、説明を読まずに分かる。
- “今” のインタラクティブ要素だけが、自然に目立つ。

### 原則 2: 反応は即時、結果は気持ちよく

- 入力（タップ/クリック/ドラッグ）に対して、**即座に** 反応（押し込み・点灯・音）を返す。
- 結果（配置・奪取・連鎖）は 1〜2 秒の範囲で “読める” 演出にする。

### 原則 3: 情報の優先順位を固定し、視線移動を減らす

- 重要: 次アクション、現在手番、スコア
- 次点: 残り枚数、アクティブなルール
- 補助: 詳細 HUD（勝率/優勢など）は必要な時だけ。

### 原則 4: 見た目は「触れそうか」で決める

- 押せる UI は必ず“押せそう”に。
  - 厚み、縁、ハイライト、影、押下時の沈み込み。
- 押せない UI は “押せそうに見せない”。

### 原則 5: 失敗しても嫌な気持ちにしない

- 置けない/選べない場合は、早めに理由を示し、やり直しを許す。
- ミス誘発の代表: 当たり判定が小さい、押下時にレイアウトが動く、状態が曖昧。

### 原則 6: 例外を作らない

- ボタン、セル、カード、パネルは共通のルール（色/影/押下）で統一する。
- “画面ごとに触り方が違う” を作らない。

### 原則 7: 演出は「情報」でもある

- アニメは飾りではなく、状態遷移を伝えるために使う。
- やりすぎると読めない。控えめでも “手触り” は作れる。

## 2) バトル盤面のチェックリスト（Nintendo 品質の最低ライン）

### A. 分かりやすさ（迷い防止）

- [ ] 手番が 1 つの場所で明確（プレイヤー側の強調、または HUD の明示）
- [ ] 次アクションが 1 行で明確（例: 「カードを選んでください」）
- [ ] 置けるセルだけが自然に目立つ（過剰な点滅ではなく、光/陰影で）
- [ ] 選んだカード・選んだセルが一目で分かる（色/縁/影/サイズの統一）

### B. 押した感（インタラクション）

- [ ] 押下で沈み込み（translate + shadow の変化）
- [ ] 離した時に少し戻る（オーバーシュートを 1 回だけ）
- [ ] タップ時に音（必要なら）とミニフラッシュ（必要なら）

### C. 読みやすさ（情報密度）

- [ ] 盤面が主役（余計な枠線や文字が盤面より目立たない）
- [ ] スコア/ターンが “探さず見える”
- [ ] ルールは小さくても“見える形”で提示（チップ/バッジ）

### D. 安定性（レイアウトが崩れない）

- [ ] Nyano コメント/通知でレイアウトが押し広げられない（absolute/overlay で描画）
- [ ] 盤面周りの高さが、表示内容で上下しない（最小高さ/固定枠）

### E. 低負荷・アクセシビリティ

- [ ] `prefers-reduced-motion` でアニメ/音が抑制される
- [ ] `data-vfx=off/low` で粒子/グローが抑制される
- [ ] 低スペック端末でも FPS が落ちにくい（blur 多用を避ける）

## 3) 実装ガイド（Mint UI での現実的な作り方）

### レイヤー設計（おすすめ）

1. 背景（空グラデ + 肉球/きらめきの薄レイヤー）
2. ボードトレイ（大きなガラス板。内側に凹みとハイライト）
3. セル（カードスロット。凹み/縁/斜めシーン）
4. カード（影 + 置いた時の沈み込み）
5. HUD（スコア/ターン/ルールチップ）
6. エフェクト（スパークル、リップル。`data-vfx` で段階制御）

### 触感の基本レシピ

- 通常: 影があり、縁が明るく、中心が少し暗い
- Hover/Focus: わずかに持ち上がる、縁が明るくなる
- Press: 少し沈む、影が短くなる、中心がほんの少し暗くなる
