# QA: Motion Language チェックリスト v1

このチェックリストは Motion Language（かわいさの動き）を入れた後に、
**テンポ/可読性/アクセシビリティ/性能**が崩れていないかを短時間で確認するためのものです。

## 事前準備

- OS/ブラウザの `prefers-reduced-motion` を切り替えられる状態にする
- VFXティア（`data-vfx`）を変えて確認する
  - ローカル設定: `localStorage["nytl.vfx.quality"]` を `off|low|medium|high|auto` に変更
  - 画面更新後に `<html data-vfx="...">` になっていることを確認

## 共通観点（全ページ）

- [ ] 押した瞬間に反応が返る（0.1秒前後）
- [ ] 動きが「意味」を増やしている（装飾で操作を妨げない）
- [ ] レイアウトが揺れない（Layout Shift が起きない）
  - `transform/opacity` 中心で、`width/height/top/left` のアニメを避ける
- [ ] `data-vfx=off` でも情報が欠けない（輪郭/影/色で置換）
- [ ] `prefers-reduced-motion: reduce` で過剰モーションが抑制される

## 主要導線

### Home → Arena → Match
- [ ] 画面遷移が速い（入場だけ強調・退出は短い）
- [ ] 主要ボタンに統一された押下感（MOT-01 むにゅ）
- [ ] 確定アクションに「カチッ」（MOT-02）が入っている

### Toast / Popover / Dialog
- [ ] ふわっと出て、引っかからずに消える（MOT-03 / MOT-05）
- [ ] `reduce` のとき、出現がフェード中心へ置換される

### Deck / Rulesets
- [ ] 切替（タブ/モード）が同じテンポ（MOT-04）
- [ ] トグルの ON/OFF が一瞬で分かる（影/色/位置）

## Match（盤面）

### 置けるマス（Magnet）
- [ ] 「置ける」が強調されるが、常時点灯しない（選択中のみ）
- [ ] 置けない時のNGが短い（MOT-12）

### 配置（Drop）
- [ ] 置いた瞬間に質量感（落下+着地）がある（MOT-09）
- [ ] 着地がテンポを壊さない（0.2〜0.3秒内）

### 奪取（Flip / Impact）
- [ ] 反転が“分かる”が、長すぎない（MOT-13）
- [ ] 確定に短い重さ（MOT-10）

### 連鎖（Chain）
- [ ] Same/Plus 連鎖の対象が順番に分かる（MOT-14）
- [ ] `data-vfx=low/off` では線演出を無効化しても意味が伝わる

## 性能

- [ ] モバイル幅でスクロール/タップが重くならない
- [ ] 連続試合で FPS が極端に落ちない（高負荷端末では `auto→low` に落ちる想定）

---

このチェックを満たしたら、Motionを“広げる”のではなく **「差分の見え方」を研磨**してください。
（例：Dropの着地1フレームだけ強く、Flipの中間フレームを短く、など）
